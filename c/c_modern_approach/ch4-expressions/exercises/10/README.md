# Exercise 3.10

Show the output produced by each of the following program fragments.
Assume i and j are int variables.

1. i = 6;
	 j = i += i;
	 `printf("%d %d", i, j);`
2. i = 5;
	 j = (i -= 2) + 1;
	 `printf("%d %d", i, j);`
3. i = 7;
	 j = 6;
	 `printf("%d %d", i, j);`
4. i = 2; j= 8;
	 j = (i = 6) + (j = 3);
	 `printf("%d %d", i, j);`

## Solutions

1. `12 12`
2. `3 4`
3. `2 9`
4. `6 9`
